Privacy Policy

Lingalyze – Chrome Extension

Overview

Lingalyze is a browser extension that provides instant translation and AI-powered grammar improvement in the browser sidebar. It can translate between languages using Google Translate or an optional AI provider, and it can analyze and improve your writing with detailed corrections. This privacy policy explains what data is collected, how it is used, and how it is stored.

Data Collection and Storage

The extension stores the following data locally on the user's device using Chrome's chrome.storage.local API:

  • User preferences: Settings such as the default target language, AI provider, and selected model are stored locally.
  • API key: If the user configures an AI provider (OpenAI or Anthropic), their API key is stored locally on the device. The key is never transmitted to the extension developer or any server other than the chosen AI provider.

The extension does not store any translation history, input text, or output text. All text processing results are displayed in the sidebar and discarded when the panel is closed or new text is entered.

Third-Party Services

Lingalyze connects to the following third-party services to provide its core functionality:

  • Google Translate (translate.googleapis.com): Used for automatic language detection and free translation. The text entered by the user is sent to Google's translation API. This is the default translation method and does not require an API key.
  • OpenAI (api.openai.com) or Anthropic (api.anthropic.com): If the user configures an AI provider, input text is sent to the selected provider for higher-quality translation or grammar improvement. The user's own API key is used for authentication. This is entirely optional.

The extension developer does not operate or control these third-party services. Please refer to their respective privacy policies for details on how they handle data:

  • Google Privacy Policy
  • OpenAI Privacy Policy
  • Anthropic Privacy Policy

Data Storage

  • User settings and API key are stored locally on the user's device using Chrome's storage API.
  • No input text, translation results, or grammar corrections are persisted after use.
  • No user data is sent to the extension developer or any server controlled by the extension developer.
  • No analytics, tracking, or telemetry data is collected.

Data Sharing

The extension does not sell, share, or transfer user data to any third party beyond what is described above. The only external communication occurs when the user initiates a translation or grammar improvement, in which case the input text is sent to Google Translate and/or the selected AI provider (OpenAI or Anthropic).

Permissions

The extension requests the following browser permissions:

  • sidePanel: Required to display the translation and grammar improvement interface in the browser sidebar.
  • storage: Required to save user settings and the API key locally on the device.
  • Host permissions (translate.googleapis.com, api.openai.com, api.anthropic.com): Required to send requests to the translation and AI services.

User Control

  • Users can remove their API key at any time from the Settings tab.
  • AI-powered features are entirely optional — the extension works with Google Translate alone.
  • No input text or results are stored; closing the sidebar or entering new text discards previous results.
  • Uninstalling the extension removes all locally stored data.
