Privacy Policy

Holy Jiaobei! – Chrome Extension

Overview

Holy Jiaobei! is a browser extension that simulates the traditional Taiwanese folk practice of throwing jiaobei (筊杯) and drawing fortune sticks (籤詩) from Tu Di Gong (土地公). It optionally uses AI-powered services to interpret fortune results. This privacy policy explains what data is collected, how it is used, and how it is stored.

Data Collection and Storage

The extension stores the following data locally on the user's device using Chrome's chrome.storage.local API:

  • Fortune history: Fortune results, user-entered questions, and AI interpretations are saved locally so users can review past readings.
  • User preferences: Settings such as interface language, AI provider, and selected model are stored locally.
  • API key: If the user enables AI interpretation, their API key (for OpenAI or Anthropic) is stored locally on the device. The key is never transmitted to the extension developer or any server other than the chosen AI provider.

Third-Party Services

Holy Jiaobei! offers an optional AI fortune interpretation feature. When the user chooses to use this feature, the following data is sent to the selected AI provider:

  • OpenAI (api.openai.com) or Anthropic (api.anthropic.com): The fortune stick number, fortune level, fortune poem text, and the user's question are sent to the provider's API to generate an interpretation. The user's own API key is used for authentication.

The extension developer does not operate or control these third-party services. Please refer to OpenAI's Privacy Policy and Anthropic's Privacy Policy for details on how they handle data.

No data is sent to any third-party service unless the user explicitly clicks the "AI Interpret" button. Core functionality (throwing jiaobei and drawing fortune sticks) works entirely offline.

The extension includes an optional donation button that opens a PayPal page (paypal.me) in a new browser tab. This action does not transmit any extension data — it simply navigates to an external URL.

Data Storage

  • All user data (history, settings, API key) is stored locally on the user's device using Chrome's storage API.
  • No user data is sent to the extension developer or any server controlled by the extension developer.
  • No analytics, tracking, or telemetry data is collected.

Data Sharing

The extension does not sell, share, or transfer user data to any third party. The only external communication occurs when the user explicitly requests an AI interpretation, in which case fortune data and the user's question are sent to the selected AI provider (OpenAI or Anthropic) using the user's own API key.

Permissions

The extension requests the following browser permissions:

  • sidePanel: Required to display the fortune-telling interface in the browser sidebar.
  • storage: Required to save fortune history, user settings, and the API key locally on the device.

User Control

  • Users can delete individual fortune history entries or clear all history at any time from the History tab.
  • Users can remove their API key at any time from the Settings tab.
  • AI interpretation is entirely optional — the extension's core functionality works without it.
  • Uninstalling the extension removes all locally stored data.
